Lucky Lua

A CLI tool for gambling written in lua by JameDev for Siege Week 1

Download

Feel free to download the bundled lua file under the latest release and run it via: 


lua lucky.lua

Video Preview:

Running on Replit

You can also run the project on replit by "remixing" this project and run it in replits cli. This is not the recommended way of testing the programm! Other online lua interpeters may work, but in my experience they either don't support live input, don't show colors or don't clear the console properly.
